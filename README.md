# Waifuset - 文生图数据集管理工具

Waifuset 是一个专门用于构建和管理文生图数据集的工具，提供安全、高效、便捷的 Web 界面。

## 特点
- **安全**：不修改图像文件，确保数据完整性
- **高效**：轻松处理十万乃至百万量级的大型数据集
- **易用**：UI 设计简洁直观，一键完成复杂操作
- **远程访问**：支持本地或云服务器部署

## 一键启动

### Windows
`ash
run.bat path/to/folder
`

### Linux/Mac
`ash
./run.sh path/to/folder
`

**就这么简单！** 脚本会自动：
- 创建 uv 虚拟环境
- 安装所有依赖
- 启动 Web 界面

## 访问界面

启动后访问：http://127.0.0.1:7860

## 项目结构

- un.bat / un.sh - 一键启动脚本
- pi.py - 主程序
- equirements.txt - 依赖列表
- waifuset/ - 核心代码

## 局限性

- 基于 Danbooru 标签系统
- 标注格式：英文逗号分隔的标签
- 仅用于标注编辑，不包含图像编辑功能

## 适用场景

1. **数据集标注构建和清洗**
2. **数据集浏览和管理**
